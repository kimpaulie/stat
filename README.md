# 통계 학습 노트 · KAIST BIT500

KAIST BIT500 *Management Statistical Analysis* 강의를 초심자를 위해 쉽게 풀어쓴 웹 교과서.

**배포**: [kimpaulie.synology.me](https://kimpaulie.synology.me)

## 수록 강의 (7강)

| 강 | 제목 | 핵심 주제 |
|----|------|----------|
| 1 | 통계 첫걸음 | 변수·데이터, 요약표·분할표, 그래프 |
| 2 | 숫자로 요약하기 | 평균·중앙값·최빈값, 분산, 왜도, 공분산·상관계수 |
| 3 | 확률의 세계 | 확률 기본, 조건부 확률, 베이즈, 독립 시각화 |
| 4 | 이산 확률분포 | 확률변수, 이항분포, 공분산 함정, 분산의 합 공식 |
| 5 | 정규분포 | 표준정규, Z-table, 다변량 정규, 호텔 체인 예제 |
| 6 | 표본분포와 CLT | 표본평균, 중심극한정리, 비율 추정 |
| 7 | 신뢰구간 | CI, t-분포, 두 평균/비율 비교, 짝 표본 |
